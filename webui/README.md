# ETA Web UI (React)

Ziel: Einfache Weboberfläche mit Seitenleiste, um verschiedene Dashboards anzuzeigen.

## Entwicklung

```
cd webui
npm install
npm run dev
# Öffnen: http://localhost:5173
```

## Build

```
npm run build
npm run preview
```
